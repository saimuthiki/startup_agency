# WhatsApp Outreach Guide

## Tool: WA Sender (SheetWA) Chrome Extension
Free extension that sends from Google Sheets via WhatsApp Web.

## Message Template
```
🙏 Namaste {business_name} ji,

I noticed your shop has excellent {rating}★ reviews on Google Maps — impressive!

I specialize in building professional websites for businesses like yours.
I've already created a sample website:

🌐 {demo_url}

A website helps your customers:
✅ Find your address and call directly
✅ See products/services with photos
✅ Contact you on WhatsApp with one tap
✅ Find you on Google Search

Special offer: ₹{price} one-time (no monthly charges first year).

Would you like a personalized version for {business_name}?
Reply "YES" and I'll build it! 🙌
```

## Anti-Ban Rules
1. Use separate SIM (₹100), not primary number
2. Warm up 1-2 weeks first
3. Start 10-15/day → scale to 30-50/day by week 3
4. 30-60 second random delays
5. Personalize every message
6. Never identical messages
7. Include "Reply STOP"
8. Business hours only (10AM-7PM IST)

## This is NOT automatable by Claude Code
WhatsApp outreach requires browser + human supervision. Use WA Sender extension.
